# Pentesting SPI

- ## Recon
    - Identify SPI Chips
    - Get Datasheets 
    - Map Pins to PCB

- ## Sniffing
    - Use Logic Analyzer
    - Tap SPI Lines
    - Sniff Unencrypted Data

- ## Memory Extraction
    - Identify Memory Chip
    - Use Flash Dumper
        - Dump Firmware
        - Extract Keys/Data

- ## Fault Injection
    - Use Glitching Tool
        - Target SPI Lines
            - Disrupt Communications 
            - Corrupt Firmware/Data

- ## Physical Tampering
    - Decapsulate Chips
    - Use Microprobing
        - Directly Access Buses

- ## Side-Channel Analysis
    - Use SCA Tools
        - Capture Power Traces
        - Extract Encryption Keys

- ## Backdooring Firmware
    - Dump Firmware
    - Modify Firmware
    - Write Backdoored Version
